# 顺时针/逆时针旋转

!!! note "版本信息"
    该特性包含于 NGDXW先辈 修改版 om_midi

[![顺时针旋转](/gallery/rotation-sample1.png)](/gallery/rotation-sample1.png)
[![逆时针旋转](/gallery/rotation-sample2.png)](/gallery/rotation-sample2.png)

每有一个音符，关键帧的值均为 `0` -> `90` -> `180` -> `270` -> `0` ...... 以此类推。

!!! note
    逆时针：`0` -> `-90` -> `-180` -> `-270`
